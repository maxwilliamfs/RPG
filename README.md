# RPG
## Descrição
Este projeto é um pequeno RPG (Roling Playing Game), desenvolvido em Java, com batalhas entre diferentes personagens jogáveis e diferentes inimigos
- Projeto em Desenvolvimento

## Funcionalidades
- Escolha de Personagem jogável
- Atributos dinamicos para as entidades (HP,Stamina,Mana)
- Diferentes inimigos
- Batalha contra a CPU

## Tecnologias e Ferramentas
- Linguagem: Java.
- Ambiente de Desenvolvimento: Linux Mint.
- Compilador: JVM.
- Versionamento: Git & GitHub.

## Execução
### Execultando pelo terminal
- Clonar o repositório e acessar a pasta:
```bash
git clone [https://github.com/maxwilliamfs/RPG.git](https://github.com/maxwilliamfs/RPG.git)  
cd src
```
- Compilar o projeto:
```bash
javac Console.java
```
- Execultar o projeto:  
```bash
java Console.java
```
