# Mini-Kernel Simulador

![Menu de Funcionalidades](/assets/screenshot)

## Descrição
Este projeto é um simulador de núcleo (Kernel) desenvolvido em C, focado em demonstrar conceitos fundamentais de Sistemas Operacionais, como gerenciamento de memória, escalonamento de processos e tratamento de estados de execução.

## Funcionalidades
- Gerenciamento de Processos (Load, Kill, ps).
- Simulação de Memória (Alocação First-Fit com vetor global).
- Interpretação de Instruções (malloc, wait, print).
- Editor de Texto Simulado (Create, Edit, Write, Delete).

## Tecnologias e Ferramentas
- Linguagem: C (Padrão C11).
- Ambiente de Desenvolvimento: Linux Mint.
- Compilador: GCC.
- Versionamento: Git & GitHub.

## Execução
### Execultando pelo terminal
- Clonar o repositório e acessar a pasta:
```bash
git clone [https://github.com/maxwilliamfs/Mini-Kernel.git](https://github.com/maxwilliamfs/Mini-Kernel.git)  
cd Mini-Kernel
```
- Compilar o projeto:
```bash
make
```
- Execultar o projeto:
```bash
./mini_kernel
```
